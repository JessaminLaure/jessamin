# Information Website

This is a simple informational website created using HTML and CSS. It provides basic information about a fictitious business or organization, including an about section, services offered, and contact details.

## Features

- Responsive design: The website is designed to be responsive and adapts to different screen sizes, making it accessible on various devices.
- Navigation bar: A navigation bar is provided to easily navigate between different sections of the website.
- Sections:
  - **About Us**: Provides a brief overview of the business or organization.
  - **Services**: Lists the services offered by the business or organization.
  - **Contact**: Displays contact information for inquiries.

## Getting Started

To view the website, simply open the `index.html` file in a web browser.

open index.html
